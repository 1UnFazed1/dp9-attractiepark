# Attractiepark – Datapunt 9

Dit project bevat een Python applicatie die een dagprogramma genereert voor een bezoeker op basis van databasegegevens.

Functionaliteiten:
- Ophalen van gegevens uit MySQL
- Genereren van een dagprogramma
- Toevoegen van horeca en souvenirwinkel
- Ophalen van temperatuur via een API
- Export naar JSON
